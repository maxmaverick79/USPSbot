date - the current time and date.

cal - displays a calendar of the current month.

df - current amount of free space

free -  display the amount of free memory, enter the free command.

exit - closing the terminal emulator, or by pressing ctrl-D

pwd - display the current working directory

ls - list the files and directories

cd /usr/bin - changed the current working directory to /usr/bin

ls -l - changed the output to the long format.
